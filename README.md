## 使用法

**このパッケージはまだ開発中です。**

`MzvHoffmanAlgebra.jl` はまだ Julia の公式パッケージレジストリに登録されていないため、GitHub 経由で直接追加します

```julia
using Pkg
Pkg.add(url="https://github.com/YuttariKanata/MzvHoffmanAlgebra.jl")
```

これで Julia のパッケージ環境に `MzvHoffmanAlgebra` が追加されます

インストール後に以下を実行するとパッケージが使えます

```julia
using MzvHoffmanAlgebra
```
